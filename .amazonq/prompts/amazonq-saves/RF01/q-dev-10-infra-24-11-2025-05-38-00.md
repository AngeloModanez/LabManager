## 🏗️ Plano de Infraestrutura Recomendado

### 1. **Estrutura de Pastas Ideal**
```
/LabManager
├── /backend          # Node.js/Express
├── /frontend         # React/Vite
├── /mobile           # Expo (fora do Docker)
├── /infra            # Docker Compose + configs
└── /docs             # Documentação
```

### 2. **Estratégia Docker**
- ✅ **Backend**: Docker (ideal para APIs)
- ✅ **Frontend**: Docker (build + serve)
- ❌ **Mobile**: Fora do Docker (Expo precisa de USB/emulador)

### 3. **Mobile fora do Docker - Por quê?**
- Expo precisa acessar dispositivos físicos via USB
- Emuladores precisam de acesso ao sistema host
- Hot reload funciona melhor localmente
- Debugging é mais eficiente

